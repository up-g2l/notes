Problem 1 — Joint PMF of Discrete Random Variables \(X\) and \(Y\)

The joint PMF is given by the following table:

\(P(X{=}x,Y{=}y)\) \(Y=1\)\(Y=2\)\(Y=3\) \(P(X=x)\)
\(X=1\) 0.10.10.2 0.4
\(X=2\) 0.20.10.1 0.4
\(X=3\) 0.10.10.0 0.2
\(P(Y=y)\) 0.4 0.3 0.3 1.0
(a) Marginal distributions of \(X\) and \(Y\)

Row sums give the marginal PMF of \(X\); column sums give the marginal PMF of \(Y\).

\[ P(X=1)=0.1+0.1+0.2=0.4,\quad P(X=2)=0.2+0.1+0.1=0.4,\quad P(X=3)=0.1+0.1+0.0=0.2. \] \[ P(Y=1)=0.1+0.2+0.1=0.4,\quad P(Y=2)=0.1+0.1+0.1=0.3,\quad P(Y=3)=0.2+0.1+0.0=0.3. \]

Marginal of \(X\): \(\;P(X=1)=0.4,\quad P(X=2)=0.4,\quad P(X=3)=0.2.\)
Marginal of \(Y\): \(\;P(Y=1)=0.4,\quad P(Y=2)=0.3,\quad P(Y=3)=0.3.\)
(b) Find \(P(X > Y)\)

List all pairs \((x,y)\) with \(x > y\) and sum their joint probabilities: \[ (2,1),\quad (3,1),\quad (3,2). \] \[ P(X>Y) = P(2,1)+P(3,1)+P(3,2) = 0.2 + 0.1 + 0.1 = 0.4. \]

\(\displaystyle P(X > Y) = 0.4\)
(c) Find \(P(X=2 \mid Y=1)\)

Use the definition of conditional probability: \[ P(X=2 \mid Y=1) = \frac{P(X=2,\,Y=1)}{P(Y=1)} = \frac{0.2}{0.4} = 0.5. \]

\(\displaystyle P(X=2 \mid Y=1) = \frac{0.2}{0.4} = 0.5\)
(d) Are \(X\) and \(Y\) independent?

\(X\) and \(Y\) are independent if and only if \(P(X=x,Y=y) = P(X=x)\cdot P(Y=y)\) for every pair \((x,y)\).

Check one cell — say \((x,y)=(1,3)\): \[ P(X=1)\cdot P(Y=3) = 0.4 \times 0.3 = 0.12, \] \[ P(X=1,\,Y=3) = 0.2. \] Since \(0.12 \ne 0.2\), the factorization fails.

\(X\) and \(Y\) are not independent, because \(P(X=1,Y=3)=0.2 \ne 0.4\times0.3=0.12 = P(X=1)\cdot P(Y=3)\).
Problem 2 — Joint PDF \(f_{XY}(x,y) = c(x+y)\) on \((0,1)\times(0,1)\)
(a) Find the constant \(c\)

The total probability must equal 1: \[ \int_0^1\int_0^1 c(x+y)\,dx\,dy = 1. \] Evaluate the inner integral first: \[ \int_0^1 (x+y)\,dx = \left[\frac{x^2}{2}+xy\right]_0^1 = \frac{1}{2}+y. \] Then the outer integral: \[ \int_0^1\left(\frac{1}{2}+y\right)dy = \left[\frac{y}{2}+\frac{y^2}{2}\right]_0^1 = \frac{1}{2}+\frac{1}{2} = 1. \] So \(c \cdot 1 = 1\).

\(\displaystyle c = 1\), hence \(f_{XY}(x,y) = x + y\) on \((0,1)\times(0,1)\).
(b) Marginal density \(f_X(x)\)

Integrate out \(y\): \[ f_X(x) = \int_0^1 (x+y)\,dy = \left[xy + \frac{y^2}{2}\right]_0^1 = x + \frac{1}{2}, \quad 0 < x < 1. \]

By symmetry, \(f_Y(y) = y + \tfrac{1}{2}\) for \(0 < y < 1\).

\(\displaystyle f_X(x) = x + \frac{1}{2}, \quad 0 < x < 1\)
(c) Find \(P(X < 0.5,\; Y > 0.3)\)

\[ P(X < 0.5,\,Y > 0.3) = \int_0^{0.5}\int_{0.3}^{1}(x+y)\,dy\,dx. \] Inner integral (\(y\) from 0.3 to 1): \[ \int_{0.3}^{1}(x+y)\,dy = \left[xy+\frac{y^2}{2}\right]_{0.3}^{1} = \left(x + \frac{1}{2}\right) - \left(0.3x + \frac{0.09}{2}\right) = 0.7x + 0.455. \] Outer integral (\(x\) from 0 to 0.5): \[ \int_0^{0.5}(0.7x+0.455)\,dx = \left[\frac{0.7x^2}{2}+0.455x\right]_0^{0.5} = \frac{0.7(0.25)}{2} + 0.455(0.5) = 0.0875 + 0.2275 = 0.3150. \]

\(\displaystyle P(X < 0.5,\;Y > 0.3) = 0.315\)
(d) Conditional density \(f_{X|Y}(x \mid y)\)

First find the marginal \(f_Y(y)\) (by symmetry with part (b)): \[ f_Y(y) = \int_0^1(x+y)\,dx = \frac{1}{2}+y, \quad 0 < y < 1. \] The conditional density is: \[ f_{X|Y}(x \mid y) = \frac{f_{XY}(x,y)}{f_Y(y)} = \frac{x+y}{\dfrac{1}{2}+y}, \quad 0 < x < 1. \]

Verification: \(\displaystyle\int_0^1 \frac{x+y}{y+\frac{1}{2}}\,dx = \frac{1}{y+\frac{1}{2}}\!\int_0^1(x+y)\,dx = \frac{y+\frac{1}{2}}{y+\frac{1}{2}} = 1.\checkmark\)

\(\displaystyle f_{X|Y}(x\mid y) = \frac{x+y}{y + \dfrac{1}{2}}, \quad 0 < x < 1\)
